# Movie Search App

Simple web application to search for movies and display detailed information using the OMDb API.

## Features

- Search movies by title
- Display movie poster, ratings, cast, plot, and more
- Dynamic background changes to movie poster
- Enter key support for quick searches
- Responsive design

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- OMDb API

## API

Uses the [OMDb API](https://www.omdbapi.com/) to fetch movie data. Requires a valid API key.

## Setup

1. Get a free API key from [OMDb API](https://www.omdbapi.com/apikey.aspx)
2. Replace the `api_key` variable in `script.js` with your key
3. Open `index.html` in a browser

## Usage

- Enter a movie title in the search bar
- Click "Search" or press Enter
- Movie details will be displayed with poster as background
